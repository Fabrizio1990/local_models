# Local AI Models

Repository containing Gemma models for on-device AI inference.

## Models

- `gemma3-1b-it-int4.litertlm` - Gemma 3 1B Italian model (int4 quantized)
- `gemma-3n-E2B-it-int4.part1.litertlm` - Gemma 3N E2B Italian model part 1 (int4 quantized)
- `gemma-3n-E2B-it-int4.part2.litertlm` - Gemma 3N E2B Italian model part 2 (int4 quantized)

## Usage

Download the models from the [Releases](https://github.com/Fabrizio1990/local_models/releases) page.
